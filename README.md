# DropBOT

Sistema de automação para gerenciamento de pedidos em modelo de dropshipping.

## 📌 Sobre
Este projeto tem como objetivo automatizar o fluxo completo de pedidos, desde a captura até o cumprimento no fornecedor, garantindo consistência, rastreabilidade e controle operacional.

## ⚙️ Funcionalidades
- Processamento de pedidos
- Automação de compra no fornecedor
- Atualização de rastreamento
- Gerenciamento de cancelamentos
- Monitoramento do ciclo de pedidos

## 🔄 Fluxo
1. Recebimento do pedido
2. Criação da ordem no fornecedor
3. Captura do código de rastreio
4. Atualização do status
5. Monitoramento até entrega ou cancelamento

## 🧠 Objetivo
Reduzir trabalho manual, minimizar erros operacionais e garantir previsibilidade no processo de dropshipping.
